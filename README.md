
# Segment Tree Engine (C++)

## Features
- Range Sum Query (RSQ)
- Range Minimum Query (RMQ)
- Lazy Propagation (Range Updates)
- Point Updates
- Scalable to 1e5+ elements

## Complexity
- Build: O(n)
- Query: O(log n)
- Update: O(log n)

## Build & Run
g++ main.cpp -o seg
./seg

## Highlights
- Supports range updates using Lazy Propagation
- Handles multiple query types
- Optimized memory usage (4*N array)
